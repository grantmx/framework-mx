Framework MX

Base simple responsive framework to rapidly build websites. This framework is a simple OCSS framework. It provides all the basic OCSS one needs to set the foundation to a simple responsive site. With that said, feel free to update the variables for the polish and custom styles one needs for a given project.

Base HTML

Framework MX uses HTML5 as its backbone. We use a number of HTML5 elements that make our lives easier that require the HTML5 doctype. Include it at the beginning of your projects.

<!DOCTYPE html>
<html lang="en">
  ...
</html>

Normalize and Reset

You'll notice at the top of the file we use a number or reset properties from Normalize.css, a project by Nicolas Gallagher and Jonathan Neal . Although we're not using the whole file, we are using the best parts for our needs in our option.

Grid System

The grid system for Framework MX using a 10 column percentage based system making a 1020 px default width container.

Basic grid

For a simple two column layout create a .grid as the top most parent element to ensure your flex grid and add the a number of span-*-* columns inside depending on the number of columns desired. The syntax span 1 of 2 means that this column spans or is column 1 of 2 total columns.

<div class="grid">
    <div class="span-1-2"></div>
    <div class="span-1-2"></div>
</div>

Responsive

Framework MX supports a number of media queries to support varying device widths and resolutions.

// Mobile Landscape
@media only screen and (min-width: 400px) { ... }

// Tablet Portrait
@media only screen and (min-width: 568px) { ... }

// Tablet Landscape
@media only screen and (min-width: 769px) { ... }

// Desktop
@media only screen and (min-width: 1020px) { ... }

// Desktop Wide & HD TV
@media only screen and (min-width: 1200px) { ... }
